# First Loop

Build the smallest playable Roblox version of Blub:

1. Players gather wood and stone from map nodes.
2. Players spend resources to place windmills.
3. Windmills generate gold over time.
4. Players can damage enemy players and windmills.
5. A round is fun when protecting windmills matters.

Skipped for now: cosmetics, saving, clans, shops, monetization, custom animations, and complex maps.
